Sliding Simulator


Instruções:

*Assim que o jogo começar, clique na janelinha para jogar;
*Utilize as setas para se movimentar;
*Utilize o espaço para voltar num deslizamento;
*Acerte nas costas de um inimigo para matar ele.




História:

O Sliding Simulator se trata de uma garota, que ficou presa num mundo onde ela é deve sempre deslizar e ela precisa de sua ajuda para poder escapar deste mundo.
